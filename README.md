# Hangouts
Телефонная книга с возможностью отпралять / принимать СМС-сообщения, звонить контактам

# Используемые технологии:
--**ViewModel**

--**Room**

--**Flow**

--**Coroutines**

--**Broadcast Receiver**

--**Content Resolver**
